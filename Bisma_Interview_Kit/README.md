## 🚀 Getting Started

To explore the kit, simply open `index.html` in any web browser.

- `bisma_interview_prep_java.html`: Part 2 — Comprehensive Java Theory.
- `bisma_interview_prep_stack.html`: Part 3 — Project-specific Stack Theory (Node.js, MongoDB).

- `bisma_companies_final.html`: Company Tracker & Targeted Application List.
